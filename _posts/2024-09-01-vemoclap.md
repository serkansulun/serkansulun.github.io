We present an online web application for video emotion classification, hosted on Google Colab. You can access it using the following links:

Direct: [https://colab.research.google.com/drive/1S-Nsm968-KTErbuU0qHOp7mEEzh6xOVx](https://colab.research.google.com/drive/1S-Nsm968-KTErbuU0qHOp7mEEzh6xOVx)

With re-direct: [https://serkansulun.com/app](https://serkansulun.com/app)

We also make the extracted pretrained features for the Ekman6 and VideoEmotion8 datasets on Zenodo. We additionally include a ekman6_blacklist.txt file, listing the videos which we found are wrongly labeled.
